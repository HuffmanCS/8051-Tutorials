This directory contains the source code for the 8051/52 Timer 2 tutorial video which explains how to setup and use Timer 2 and its respective interrupt in an 8052 MCU (specifically the AT89S52 from Atmel). 

timer2int.s     -C51 Assembly source file (can be assembled with c51asm assembler) 
timer2int.hex   -Hex file containing machine code assembled by c51asm assembler (assembled for AT89S52)

timer2int.c     -C source file (can be compiled with Small Device C Compiler (SDCC)
timer2int.ihx   -Intel hex file containing machine code compiled by SDCC (compiled for AT89S52)
