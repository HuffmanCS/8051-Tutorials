This directory contains the source code for the 8051/52 UART & Interrupt tutorial video which explains how to setup and use the UART interface and its respective interrupt in an 8051 MCU (specifically the AT89S52 from Atmel). 

uart.c     -C source file (can be compiled with Small Device C Compiler (SDCC)
uart.ihx   -Intel hex file containing machine code compiled by SDCC (compiled for AT89S52)
